# Predictive Maintenance for Medical Devices

This project focuses on predicting potential failures in medical devices using machine learning techniques. The goal is to support preventive maintenance and reduce unexpected device downtime in healthcare environments.

## Project Overview
- **Objective:** Predict device failure based on operational and sensor data
- **Type:** Supervised Machine Learning (Classification)
- **Tools & Libraries:** Python, pandas, numpy, scikit-learn, matplotlib, seaborn
- **Environment:** Google Colab

## Key Steps
- Data preprocessing and cleaning
- Feature selection
- Training machine learning models
- Model evaluation and performance analysis

## Outcome
The model helps identify devices at risk of failure, enabling proactive maintenance and improving patient safety.

## Skills Demonstrated
- Data preprocessing
- Machine learning modeling
- Healthcare-related analytics
- Predictive maintenance concepts
